## ¿Qué es CSS?

CSS es un acrónimo de *Cascading Style Sheets*, lo que significa que CSS es un lenguaje de 
estilos que se utiliza para definir la presentación del documento.

Consideremos a HTML como la estructura del documento y del contenido, y CSS es la apariencia del mismo.

A partir de este punto, necesitaremos un archivo HTML de ejemplo para aplicar los conceptos CSS que iremos viendo.
