Si estás familiarizado con Python, es posible que hayas notado que la función integrada `print()` 
no necesita un carácter `\n` explícito al final de cada línea. 
Esto significa que la salida del siguiente fragmento de código

```python
print('Curriculum')
print('News')
print('YouTube Channel')
```

será:

```
Curriculum
News
YouTube Channel
```

Esto sucede porque la función `print()` agrega el carácter `\n` al final de cada línea impresa. 
Aunque la mayoría de las veces esto es una conveniencia, es posible que a veces quieras imprimir 
líneas sin el carácter `\n`. 

En este artículo, veremos cómo cambiar el final de cada instrucción de impresión. 
Y también aprenderás otras formas de modificar el comportamiento predeterminado de la función integrada.

## Cómo imprimir sin una nueva línea en Python

Según la documentación oficial de la función `print()`, la firma de la función es la siguiente:

```python
print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)
```

Como puedes ver, la función toma cinco argumentos en total. El primero son los objetos que deseas imprimir en la terminal. 
Luego está el separador (sobre el que aprenderá en una sección posterior) y el parámetro `end`. 

El valor predeterminado del parámetro `end` es `\n`, lo que significa que se agregará un carácter de nueva línea 
a cada línea impresa en el terminal.

Para cambiar este comportamiento, simplemente puedes anular este parámetro de la siguiente manera:

```python
print('Curriculum', end=', ')
print('News', end=', ')
print('YouTube Channel')
```

Dado que cambié el valor de `end` a una coma, la salida será `Curriculum, News, YouTube Channel` en lugar de las tres líneas separadas. 
Puedes utilizar cualquier carácter como valor de `end`. 
Si no quieres nada al final de las líneas, simplemente usa una cadena vacía.

## Cómo imprimir con un separador en Python

¿Recuerdas el argumento `sep` en la firma de la función? En esta sección, aprenderás sobre su uso. 

Es posible que ya sepas o no que la función `print()` puede tomar varios objetos a la vez para imprimirlos de la siguiente manera:

```python
print('Curriculum', 'News', 'YouTube Channel')
```

La salida de este código será:

```
Curriculum, News, YouTube Channel
```

Como puedes ver, la función ha impreso las tres cadenas en una sola línea separadas por comas. 
Si deseas usar algo más como un guión como separador, puedes hacerlo de la siguiente manera:

```python
print('Curriculum', 'News', 'YouTube Channel', sep=' - ')
```

La salida de este código será:

```
Curriculum - News - YouTube Channel
```

Al igual que el argumento `end`, puedes usar más o menos cualquier cadena válida como valor del argumento `sep`.

## Cómo imprimir en un archivo en Python

En lugar de imprimir cosas en la terminal, también puedes usar la función `print()` para imprimir cosas directamente en un archivo. 

La función `print()` toma otro argumentos `file` que por defecto es `sys.stdout` o la salida estándar. 
Es el descriptor de archivo predeterminado donde un proceso puede escribir la salida. 

Puedes anular esto para escribir en un archivo en su lugar de la siguiente manera:

```python
with open('output.txt', 'w') as f:
    print('un texto de prueba', file=f)
```

Primero, abre un archivo llamado `output.txt` como `f` y lo pasa como el valor del argumento `file`. 
Si ejecutas el código, se creará un nuevo archivo y dentro de ese archivo estará su salida.

## Conclusión

La función `print()` es una de las funciones más utilizadas en Python. Por lo tanto, tener una buena comprensión de los diferentes 
usos de esta función integrada puede aumentar tu productividad. 

También hay otro argumento, `flush`, en la función. Es un booleano y establecerlo en `True` vaciará la salida en cada ejecución.
